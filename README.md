---

# Жоба туралы

**SmartMektep** — орта мектептегі оқу процесін автоматтандыруға арналған веб-негізделген ақпараттық жүйе.

Жоба дипломдық жұмыс аясында әзірленген және мектептің негізгі бизнес-процестерін цифрландыруға бағытталған.

Жүйе оқушылар, мұғалімдер, ата-аналар және мектеп әкімшілігі арасындағы өзара байланысты ұйымдастыруға мүмкіндік береді. Барлық деректер орталықтандырылған түрде сақталып, пайдаланушылар өз рөлдеріне сәйкес қажетті ақпаратқа қол жеткізе алады.

Сонымен қатар жүйеде жасанды интеллект негізіндегі көмекші енгізілген. AI модулі пайдаланушының рөлі мен сұранысына байланысты интеллектуалды көмек көрсетіп, қажетті ақпаратты жылдам табуға мүмкіндік береді.

# Жобаны іске қосу нұсқаулығы

Төмендегі қадамдарды ретімен орындаңыз.

---

# 1-қадам. Архивті ашу

Архивті компьютеріңізге жүктеп алып, кез келген бумаға шығарыңыз.

Мысалы:

```text
D:\Projects\SmartMektep
```

немесе

```text
C:\SmartMektep
```

---

# 2-қадам. Backend тәуелділіктерін орнату

Backend папкасына өтіңіз.

Windows жүйесінде:

1. backend папкасын ашыңыз
2. Бос жерге тышқанның оң жақ батырмасын басыңыз
3. "Copy as path" таңдаңыз

Мысалы:

```text
D:\SmartMektep\backend
```

CMD немесе PowerShell ашып:

```bash
cd "D:\SmartMektep\backend"
```

командасын орындаңыз.

Кейін қажетті кітапханаларды орнатыңыз:

```bash
npm install
```

Орнату аяқталғанша күтіңіз.

---

# 3-қадам. Frontend тәуелділіктерін орнату

Жаңа CMD немесе PowerShell терезесін ашыңыз.

frontend папкасының жолын көшіріп алыңыз.

Мысалы:

```text
D:\SmartMektep\frontend
```

Оған өтіңіз:

```bash
cd "D:\SmartMektep\frontend"
```

Тәуелділіктерді орнатыңыз:

```bash
npm install
```

Орнату аяқталғанша күтіңіз.

---

# 4-қадам. Backend серверін іске қосу

Бірінші терминалда backend папкасында тұрғаныңызға көз жеткізіңіз.

Келесі команданы орындаңыз:

```bash
node src/server.js
```

Егер сервер сәтті іске қосылса, консольде сервердің іске қосылғаны туралы хабарлама шығады.

Мысалы:

```text
Server running...
Database connected...
```

Осы терезені жаппаңыз.

Backend жұмыс істеп тұруы керек.

---

# 5-қадам. Frontend қосымшасын іске қосу

Екінші терминалда frontend папкасында тұрғаныңызға көз жеткізіңіз.

Келесі команданы орындаңыз:

```bash
npm run dev
```

Сәтті іске қосылған жағдайда консольде келесіге ұқсас сілтеме пайда болады:

```text
Local: http://localhost:5173/
```

---

# 6-қадам. Жүйеге кіру

Консольде шыққан сілтемені браузерге енгізіңіз.

Мысалы:

```text
http://localhost:5173
```

Жүйенің авторизация беті ашылады.
